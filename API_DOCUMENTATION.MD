# API Documentation

This section outlines the available API endpoints for interacting with the dChange platform.

## Endpoints

### 1. **GET /assets**
- **Description**: Retrieves a list of available assets on the platform.
- **Response**:
  ```json
  [
    {
      "assetId": "1",
      "name": "Bitcoin",
      "symbol": "BTC"
    },
    ...
  ]
